# Code Style Guide

## Language: Python

### Naming Conventions

- Functions/variables: snake_case
- Classes: PascalCase
- Constants: UPPER_CASE

### Formatting

- Use Black formatter
- Max line length: 88 characters
- Use type hints

### Best Practices

- Use list comprehensions
- Prefer f-strings
- Use context managers
